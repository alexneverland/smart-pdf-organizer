# Smart PDF Organizer 📄🤖

Ένα επαγγελματικό εργαλείο Python για την αυτόματη οργάνωση και ταξινόμηση σκαναρισμένων PDF (Τιμολόγια, Παραγγελίες, Αποδείξεις), χρησιμοποιώντας τεχνολογία OCR και έξυπνους κανόνες.

## Δυνατότητες ✨

- **Αυτόματο OCR:** Αναγνωρίζει κείμενο από σκαναρισμένες εικόνες/PDF χρησιμοποιώντας το Tesseract.
- **Έξυπνη Ταξινόμηση:** Καταλαβαίνει το είδος του εγγράφου βάσει λέξεων-κλειδιών (keywords) που ορίζει ο χρήστης.
- **Δυναμική Μετονομασία:** Μετονομάζει τα αρχεία αυτόματα με βάση την Ημερομηνία, τον Τύπο και τον Αριθμό Παραστατικού.
- **Graphical User Interface (GUI):** Εύχρηστο περιβάλλον με καρτέλες για Ρυθμίσεις, Κανόνες και Εκτέλεση.
- **Λειτουργία Προσομοίωσης (Dry Run):** Δείτε πώς θα οργανωθούν τα αρχεία σας πριν γίνει η πραγματική μετακίνηση.
- **Ρυθμίσεις Paths:** Ορισμός των διαδρομών για Tesseract και Poppler μέσα από την εφαρμογή.

## Απαιτήσεις Συστήματος 🛠️

1. **Python 3.x**: [Κατεβάστε την από εδώ](https://www.python.org/downloads/)
2. **Tesseract OCR**: [Κατεβάστε το από εδώ](https://github.com/UB-Mannheim/tesseract/wiki)
   - *Σημείωση:* Κατά την εγκατάσταση, εγκαταστήστε και τα Ελληνικά (Greek Language Data).
3. **Poppler for Windows**: [Κατεβάστε το από εδώ](https://github.com/oschwartz10612/poppler-windows/releases/)
   - *Οδηγία:* Κατεβάστε το zip, κάντε αποσυμπίεση και κρατήστε τον φάκελο `bin`.

## Εγκατάσταση ⚙️

1. Ανοίξτε το τερματικό (CMD ή PowerShell) στον φάκελο του project.
2. Εγκαταστήστε τις απαραίτητες βιβλιοθήκες:
   
   pip install pytesseract pdf2image pdfplumber

## Οδηγίες Χρήσης 🚀

### Βήμα 1: Πρώτη Εκτέλεση & Ρυθμίσεις
Τρέξτε την εφαρμογή με την εντολή:

    python gui.py

Πηγαίνετε στην καρτέλα **"⚙ Ρυθμίσεις"**:
1. Στο πεδίο **Tesseract**, επιλέξτε το αρχείο `tesseract.exe` (συνήθως στο `C:\Program Files\Tesseract-OCR`).
2. Στο πεδίο **Poppler**, επιλέξτε τον φάκελο `bin` που κατεβάσατε (π.χ. `C:\poppler\Library\bin`).
3. Πατήστε **"💾 Αποθήκευση Ρυθμίσεων"**.

### Βήμα 2: Δημιουργία Κανόνων (Groups)
Πηγαίνετε στην καρτέλα **"🧩 Κανόνες"**:
1. Πατήστε **"+"** για να προσθέσετε μια νέα κατηγορία (π.χ. "Τιμολόγια Προμηθευτών").
2. Δώστε ένα **Prefix** (π.χ. "TIMOL").
3. Γράψτε τις **Λέξεις Κλειδιά** που υπάρχουν μέσα στα έγγραφα (μία ανά γραμμή), π.χ.:
   - ΤΙΜΟΛΟΓΙΟ
   - INVOICE
   - ΔΕΛΤΙΟ ΑΠΟΣΤΟΛΗΣ
4. Πατήστε **"💾 Αποθήκευση JSON"**.

### Βήμα 3: Οργάνωση Αρχείων
Πηγαίνετε στην καρτέλα **"📂 Οργάνωση"**:
1. Επιλέξτε τον **Φάκελο Εισαγωγής** (εκεί που έχετε τα ακατάστατα PDF).
2. Επιλέξτε τον **Φάκελο Ταξινόμησης** (εκεί που θέλετε να πάνε).
3. (Προαιρετικά) Τσεκάρετε το **"Λειτουργία Προσομοίωσης (Dry Run)"** για να κάνετε δοκιμή χωρίς φόβο.
4. Πατήστε **"▶ ΕΚΚΙΝΗΣΗ"**.
